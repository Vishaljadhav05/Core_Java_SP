# 🚀 How to Run Java Programs

1️⃣ Install **JDK** 

2️⃣ Clone the repository

```bash
git clone https://github.com/your-username/core-java-learning.git
```

3️⃣ Navigate to any folder and compile the program

```bash
javac ProgramName.java
```

4️⃣ Run the program

```bash
java ProgramName
```
